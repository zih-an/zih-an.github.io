    + 反函数：$x=f^{-1}(y)$
        1. 严格单调函数必有反函数。：1个y只能对应1个x

    + 有界性：$|f(x)|\le M$。：必须指明区间；只要这个区间上任意一点$x_0$使$\lim\limits_{x\to x_0}f(x)=\infty$，则无界

## 2 数列的极限
> 极限是一个过程，$\epsilon$是一个“差不多”的思想
> <img src="2数列极限.JPG">
> - 定义、性质、夹逼准则、单调有界准则：证数列极限问题

### 2.1 定义
$\epsilon - N$ 语言：注意，数列中的$\infty$特指正无穷（n是自然数）
- 用定义法证明极限
    1. $|x_n-A| < \epsilon$
    2. 反解 $x_n=g(\epsilon)$
    3. $N=\lceil g(\epsilon)\rceil$
- 数列收敛，任何一个子数列都收敛，且极限相同。
    + 逆否命题常用于证明 发散

### 2.2 性质
1. 唯一性
2. 有界性：极限存在必有界。反之不成立，反例$\mathop{sin}x_n$有界无极限
3. 保号性（脱帽）：$\lim\limits_{n\to \infty} x_n = A$，若$A>0(或A<0)$，则$n>N时，有x_n>0(或x_n<0)$。（即使$A\ge 0(或A\le 0)$，仍然$有x_n>0(或x_n<0)$，取不到等号。
    + 推论（带帽）：$若从某项起有x_n\ge 0，且极限存在为A，则A\ge 0$。能取到等号

### 2.3 四则运算
！只有**有限个**数列相运算才可以使用四则运算规则
- 对于无限个数列，考虑夹逼准则
- 无穷级数研究无限个数列

### 2.4 夹逼准则
不验证等号
- 无限个数列，考虑夹逼准则

### 2.5 单调有界准则 （很重要）
单调有界数列必有极限。（证单调性+有界性。有界性的证明，充满了不等关系）
- 若数列${a_n}$是单调递增（递减）数列，且有上界（下界），则数列${a_n}$必有极限
    + 单增考虑上界，反之亦然。
    + 单减考虑下界，反之亦然。
- 形如$a_{n+1}=ka_n$类似的数列递推式，考虑使用单调有界准则

### Conclusion
> 此节内容针对数列，侧重于极限的证明。并且**充满了不等关系**（各种不等关系很重要）
> 1. 定义法证明：没有什么解析式，看起来比较干，就定义法证吧
> 2. 性质证明（用唯一、有界、保号、子数列关系）：更侧重于发散的证明（各种逆否命题的应用）
> 3. 夹逼准则：侧重于“无限个数列”（例如求和一类），无法使用四则运算规则，典型的需要放缩
>       - 放缩有技巧：naive放缩，尽量只动分母，不动分子；非naive，保证“公比”相同
> 4. 单调有界准则：侧重于递推式的证明
>       - 单增考虑有上界，反之亦然
>       - 单减考虑有下界，反之亦然
> 
> 进一步：
> - 证明发散：1定义法；2性质（唯一性、有界性、保号性、与子数列关系）
> - 证明收敛：1定义法；2夹逼；3单调有界
> - 求值：1定义法；2夹逼；3四则运算；4单调有界证明后，两端直接套lim求解
> - 都考虑需要结合四则运算

## 3 函数的极限
定义：$\epsilon-\sigma$语言，$\epsilon-X$语言
### 3.1 计算极限（函数）
- 七种不定型

| 极限值的形式 | 解法 | 
| :---: | :---: |
| $\frac{0}{0}$ | 洛必达、泰勒 |
| $\frac{\infty}{\infty}$ | 洛必达、泰勒 |
| $0\cdot \infty$ | 除法转以上 |
| $0^0$ | 用对数转以上 |
| $\infty^{0}$ | 用对数转以上 |
| $1^{\infty}$ | 用对数转以上 |

- 其他易与上混淆型（都是定型）

| 极限值的形式 | 解 | 
| :---: | :---: |
| $\frac{0}{\infty}$ | $0$ |
| $\frac{\infty}{0}$ | $\infty$ |
| $0\cdot 0$ （有限个连乘可） | $0$ |
| $0\cdot c$ （有限个连乘可） | $0$ |
| $\infty\cdot \infty$ | $\infty$ |
| $c\cdot \infty$ | $\infty$ |
| $0^{\infty}$ | $0$ |
| $\infty^{\infty}$ | $\infty$ |
| $1^0$ | 1 |

- 其他

| 极限值的形式 | 解 | 
| :---: | :---: |
| $\frac{a}{b}$ | $\frac{a}{b}$ |
| $\frac{\infty}{c}$ | $\infty$ |
| $\frac{c}{\infty}$ | $0$ |
| $\frac{0}{c}$ | $0$ |
| $\frac{c}{0}$ | $\infty$ |

- 泰勒展开（需熟记部分）

| 函数 | 展开 | 变形 |
| :---: | :---: | :---: |
| $\mathop{sin}x$ | $x-\frac{1}{3!}x^3+o(x^3)$ |
| $\mathop{arcsin}x$ | $x+\frac{1}{3!}x^3+o(x^3)$ |
| $\mathop{cos}x$ | $1-\frac{1}{2!}x^2 + \frac{1}{4!}x^4+o(x^4)$ |
| $\mathop{tan}x$ | $x+\frac{1}{3}x^3+o(x^3)$ |
| $\mathop{arctan}x$ | $x-\frac{1}{3}x^3+o(x^3)$ |
| $\mathop{ln}(1+x)$ | $x-\frac{1}{2}x^2+\frac{1}{3}x^3+o(x^3)$ |
| $\mathop{e}^x$ | $1+\frac{1}{2!}x^2+\frac{1}{3!}x^3+o(x^3)$ |
| $(1+x)^a$ | $1+ax+\frac{a(a-1)}{2!}x^2+o(x^2)$ |
> 做题小技巧：如果分子分母阶数都待定，则先定容易的（一般先定分母展开后阶数，分子看齐）

- 四则运算条件：有限个，且函数极限均存在，即均不为$\infty$。除法中，分母不为0.
- 复合函数的极限运算法则：
    1. 选$u=g(x)$
    2. 求$\lim\limits_{x\to\cdot}g(x) = u_0$
    3. 求$\lim\limits_{x\to\cdot}f(u)$
    + 所以，遇到$e^t, \mathop{sin}t$一类的式子，可以直接先求里面，再把值$u_0$代出来
    + 变量代换同理
- 几个函数极限计算中的重要函数（要考虑左右极限）
    + $e^x$：指数函数
    + $|x|$：绝对值
    + $arctanx$：反三角函数
    + $[x]$：取整
- 连续和间断点问题考虑
    + 无定义点
    + 分段点

> 自证性定理结论（过程参考笔记）：
> 1. 当求极限的部分为乘除法时，如$\lim\limits_{x\to \cdot} f(x)\cdot g(x)$，若有$\lim\limits_{x\to \cdot}g(x)=C (C\ne 0或\infty)$，则$\lim\limits_{x\to \cdot} f(x)\cdot g(x) = C\lim\limits_{x\to \cdot} f(x)$.
>       + （自证）补充四则运算。
> 2. 若$\lim\limits_{n\to\infty}u_n=a$，则$\lim\limits_{n\to\infty}|u_n|=|a|$
>       + （习题）反之不成立。
> 3. 若奇、偶子列极限均存在，且都趋于a，则原数列极限存在且为a
>       + （习题）反之也成立，即是数列与子列的关系。
